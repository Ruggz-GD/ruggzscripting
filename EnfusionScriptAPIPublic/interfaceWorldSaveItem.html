int iQuality,
int iChannels,
bool bPreview )

Creates JPEG data from PixelRawData based on given parameters.

Parameters
pData- raw data of the image
iWidth- width of the image
iHeight- height of the image
iQuality- between 1-100, higher quality results in a bigger image
iChannels- how many channels to use (in case of JPEG its 4 as CMYK)
bPreview- true if the screenshot should be uploaded as preview, false if it should be added to gallery instead
Returns
int - 0 on failure, non-0 on success
